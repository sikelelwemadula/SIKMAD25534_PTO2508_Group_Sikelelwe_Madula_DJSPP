# 🎙️ Podcast Web Application

This is the final project in our JavaScript learning journey. We were required, using React.js, to create a Podcast App. throughout the course, we did components of this app till this point. 

A modern, responsive React web application that allows users to browse, search, and filter podcasts, manage a list of their favorite episodes with tracking timestamps, and listen to audio tracks using a persistent media player context.

## 🛠️ Technologies Used

- **Frontend Core:** React 18, JSX, JavaScript (ES6+)
- **Languages Used:** HTML, CSS, JavaScript-React
- **State Management:** React Context API (Modular contexts for Podcasts and Audio tracking)
- **Styling:** CSS Modules
- **Data Persistence:** Web Storage API (`localStorage`)
- **Build Tool:** Vite

## 🚀 Features

- **Dynamic Podcast Fetching:** Pulls comprehensive show metadata, seasons, and episode tracks seamlessly via an API client.
- **Advanced Filtering & Sorting:** Filter shows dynamically by specific genre categories, or use an instant text search bar. Sort content effortlessly by title (A-Z / Z-A) or release date (Newest / Oldest).
- **Persistent Favorites with Timestamps:** Save individual episodes to a personalized favorites dashboard. Each favorited episode logs the precise date and time it was added, persisting securely across sessions via browser `localStorage`.
- **Global Audio Player Context:** Listen to audio files continuously with an integrated global audio controller that plays music smoothly even when navigating between different views.
- **Responsive Layout Calculation:** Features a responsive card view sizing matrix that dynamically recalculates item page splits on the fly when resizing browser viewing grids.
